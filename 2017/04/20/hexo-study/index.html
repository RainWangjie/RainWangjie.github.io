init

新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

new

新建一篇文章。如果没有设置layout的话，默认使用 _config.yml 中的 default_layout参数代替。如果标题包含空格的话，请使用引号括起来。

generate

生成静态文件。

选项 描述
-d, –deploy 文件生成后立即部署网站
-w, –watch 监视文件变动

该命令可以简写为

server

启动服务器。默认情况下，访问网址为： http://localhost:4000/。

deploy

部署网站。
| 选项 | 描述|
| —–|:—:|
|-g, –generate |部署之前预先生成静态文件|

该命令可以简写为

tools
