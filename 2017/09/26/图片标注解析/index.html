图片标注解析

发表于 2017-09-26 |

简介

图片标注系统分为3个部分：鼠标动作行为识别与记录、标注框渲染、标签管理。

鼠标动作监听与渲染过程已分离，独立执行。

鼠标事件系统

鼠标事件分为3个过程：mousedown、mousemove（可不执行）、mouseup

鼠标动作行为有3种：新建、移动、缩放

1. 3个mousedown事件绑在3个元素上，确定操作行为与操作标注索引（selected）
    * newLabelMouseDown => $('#img-area-self') 新建
    * moveLabelMouseDown => $('.label-area') 移动
    * scaleLabelMouseDown => $('.ui-resizable-handle') 缩放
2. 全局mousemove => $('#img-area-self')
3. 全局mouseup => $('#img-area-self')

标注框渲染

结合requestAnimationFrame，根据浏览器刷新率（60HZ），在浏览器下一渲染前插入标注框的渲染方法（异步执行），解决mousemove事件（可能会大于60HZ）对渲染方法的影响，引起卡顿。
